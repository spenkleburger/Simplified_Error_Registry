# Max Mode

Normally, Cursor uses a context window of 200k tokens (~16,000 lines of code). Max Mode is our option to turn on the maximum context windows for all models. This will be slower and more expensive. It is most relevant for Gemini 2.5 Pro and GPT 4.1, which have 1M token context windows.

Learn more about [how context works](/learn/context) and [tokens and pricing](/learn/tokens-pricing).
