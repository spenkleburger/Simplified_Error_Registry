## Platform Support

Cursor is available for all major operating systems with native installation packages.

### Windows

Windows 10 and later
Native installer (.exe)

### macOS

macOS 10.15 (Catalina) and later
Native installer (.dmg)
Apple Silicon and Intel support

### Linux

Cursor now offers multiple installation options for Linux:

#### Debian/Ubuntu (.deb packages)

Native `apt` package manager support
Automatic updates through `apt`
Proper desktop integration
Available for both x64 and ARM64 architectures

```
# Installation via package manager coming soon
# For now, download .deb from cursor.com/downloads
```

#### RHEL/Fedora (.rpm packages)

Native `dnf`/`yum` package manager support
Automatic updates through package manager
Proper desktop integration
Available for both x64 and ARM64 architectures

```
# Installation via package manager coming soon
# For now, download .rpm from cursor.com/downloads
```

#### Universal (AppImage)

Works on any Linux distribution
Portable, no installation required
Available for both x64 and ARM64 architectures

The `.deb` and `.rpm` packages provide the best experience on Linux with proper system integration, automatic updates, and working `cursor` command-line tools.
